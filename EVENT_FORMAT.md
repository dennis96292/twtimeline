## 格式

輸出必須是一個 **JSON 陣列**，每個元素代表一筆歷史事件。

```json
[
  {
    "date": "YYYY-MM-DD",
    "title": "事件標題",
    "desc": "事件描述",
    "tags": ["標籤一", "標籤二"],
    "sources": [
      {
        "outlet": "中央社",
        "title": "這篇報導的實際標題",
        "url": "https://www.cna.com.tw/news/..."
      },
      {
        "outlet": "自由時報",
        "title": "另一篇報導的標題",
        "url": "https://news.ltn.com.tw/news/..."
      }
    ]
  }
]
```

---

## 欄位說明

| 欄位 | 型別 | 必填 | 說明 |
|------|------|------|------|
| `date` | string | 是 | 事件發生日期，格式 `YYYY-MM-DD`。若僅知年份填 `YYYY-01-01`，僅知年月填 `YYYY-MM-01` |
| `title` | string | 是 | 簡短事件名稱，建議 20 字以內 |
| `desc` | string | 否 | 事件描述，建議 50～150 字，說明事件背景、經過與影響 |
| `tags` | array | 否 | 分類標籤陣列，從下方標準標籤中選取，可複選 |
| `sources` | array | 否 | 多筆實際報導的來源物件（或單一網址字串，見向下相容說明） |

### `sources` 子欄位

| 欄位 | 型別 | 必填 | 說明 |
|------|------|------|------|
| `outlet` | string | 是 | 媒體名稱，優先使用下方常見媒體名單以獲得對應品牌色塊 |
| `title` | string | 是 | 該則報導的標題（通常與事件 `title` 不同，應是媒體實際下的標題） |
| `url` | string | 是 | 指向該則報導的實際網址（非首頁） |

### 向下相容

1. `sources` 元素也可以直接是字串網址。渲染時會根據 URL 網域自動推斷 `outlet`，並以事件 `title` 作為報導標題：

   ```json
   "sources": ["https://www.cna.com.tw/news/aipl/202401130099.aspx"]
   ```

2. **舊欄位 `source_url`**（單一字串網址）也仍被識別。自動化 agent（`agents/news_agent.py`）目前輸出此欄位；當事件沒有 `sources` 但有 `source_url` 時，渲染器會把它當作單一來源顯示。新事件或升級既有事件時請改用 `sources`；兩欄位**不必同時存在**，`sources` 優先。

不建議新資料使用這些向下相容形式。

## 標準標籤

請從以下標籤中選取，**不要自創新標籤**：

- `政治`
- `選舉`
- `兩岸`
- `外交`
- `經濟`
- `社會`
- `人權`
- `災害`
- `疫情`
- `環境`
- `歷史`

---

## 常見 `outlet` 名稱

以下媒體名稱會套用對應品牌顏色的色塊標示，建議優先使用：

`中央社`、`自由時報`、`聯合新聞網`、`中時新聞網`、`三立新聞網`、`TVBS新聞網`、`東森新聞`、`ETtoday`、`風傳媒`、`公視新聞`、`民視新聞`、`華視新聞`、`關鍵評論網`、`報導者`、`鏡週刊`、`蘋果新聞網`、`新頭殼`、`BBC`、`Reuters`、`CNN`、`紐約時報`、`Japan Times`、`維基百科`

未列於此名單的媒體仍可使用，將顯示預設橘色色塊。

## 規則

1. 整體輸出只能是 JSON 陣列，**不附加任何說明文字、markdown 區塊、程式碼區塊標記**
2. 所有字串值使用**繁體中文**（引用外文媒體標題時保留原文）
3. `date` 必須是合法日期，年份範圍 `1895-01-01` ～ 今日
4. `title` 不可與同一批資料中其他事件重複
5. `tags` 陣列不可為 `null`，無標籤時填 `[]`
6. `sources` 不可為 `null`，無來源時填 `[]`
7. `sources` 中每一筆必須是**實際存在且描述該事件**的獨立報導，不可捏造
8. 若事件橫跨多日（如颱風、疫情），填**事件開始日**或**最具代表性的單一日期**

```json
[

    "title": "2024總統大選",

    "tags": ["政治", "選舉"],
    "sources": [

        "outlet": "中央社",
        "title": "賴清德蕭美琴當選正副總統 民進黨完成三連霸",
        "url": "https://www.cna.com.tw/news/aipl/202401130099.aspx"

      {
        "outlet": "自由時報",
        "title": "賴蕭配勝出 得票率40.05%",

    ]
  },
```

## 常見錯誤（請避免）

| 錯誤 | 正確 |
|------|------|
| `"date": "1999/09/21"` | `"date": "1999-09-21"` |
| `"date": "民國88年9月21日"` | `"date": "1999-09-21"` |
| `"tags": null` | `"tags": []` |
| `"sources": null` | `"sources": []` |
| `"sources": "https://..."` | `"sources": ["https://..."]` |
| 捏造 `title`（非報導實際標題） | 從實際文章抓取或貼上原始標題 |
| `url` 填首頁而非文章頁 | 填指向單一文章的深層連結 |
| 輸出包含 \`\`\`json 區塊標記 | 直接輸出純 JSON |
| `"title": "九二一大地震九二一大地震發生於1999年..."` | title 簡短，細節放 desc |
